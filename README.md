## Reflection

Zoho Creator is a Software as a Service (SaaS) platform because it allows users to build and manage applications through a web browser without installing or maintaining software locally. The platform is hosted in the cloud, making it accessible from anywhere with an internet connection. GitHub supports assessment tracking by maintaining version history, recording every update, and enabling easy submission of project files. It also helps faculty verify progress and collaboration while ensuring secure storage of project documents. Overall, this project improved my understanding of cloud computing, SaaS platforms, and GitHub-based project management.
